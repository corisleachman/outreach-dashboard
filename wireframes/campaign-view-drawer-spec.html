campaign-view-drawer.html

Feature specification for clean rewrite · outreach. wireframe prototype · March 2026

Shell & Layout

Persistent shell

  • Fixed topbar (56px): logo left, breadcrumb Campaigns / Agency founders — Q1, plan badge + avatar right
  • Slim sidebar (56px wide): Campaigns icon active, Summary icon → summary-and-workflow.html, Settings cog (pinned bottom) → settings-account.html
  • Icons: visible at ~0.6 opacity at rest, full opacity on hover, pink tint when active. Tooltips appear on hover only (never permanently)
  • Settings icon pinned to bottom via margin-top:auto

Campaign header (sticky below topbar)

  • Campaign name (large, bold) + meta line: prospects count · start date · sequence steps · "Edit message" link
  • Stats strip: Drafts ready (pink) · Responded · Meetings set · LinkedIn sent · LinkedIn due
  • Action buttons: + Add prospects → prospect-import.html, LI message button (cyan, opens LinkedIn config modal)
  • Dev campaign switcher (bottom bar): switch between Agency founders Q1 and Fractional CFO Q1

Tab Bar — 8 tabs

Review 563 Drafts 15 LinkedIn Sent Follow-up 14 Nurture Conversations Bounced
  • Clicking a tab shows its content panel, hides all others
  • Active tab: pink text + pink bottom border
  • Search bar above the list on every tab

Tab 1 — Review

  • Header: "Prospect review" title + subtitle + Refresh button
  • 3 prospect cards: Howard Sullivan (HS), Mark Jory (MJ), Sophie Clarke (SC)
  • Each card row: avatar · name · subtitle · LinkedIn button · Skip (×) · Approve (✓) · Detail icon (↗)
  • ✓Clicking anywhere on the row (or detail icon) opens the prospect detail drawer
  • Skip button: card fades and removes from list, toast confirms
  • Approve button: card fades and removes, toast confirms
  • No catch-all or verified badges on rows (removed)

Prospect detail drawer (right panel)

  • Slides in from right (480px wide), overlays content without pushing list
  • Backdrop (dimmed overlay behind drawer) — clicking it closes drawer
  • Header: avatar · name · role · ✕ close button
  • Fields: first name (editable) · last name (editable) · email (editable) · company (editable)
  • Read-only: industry · org size · LinkedIn link
  • About section · Notes textarea
  • Email generation: when email is empty, label row shows No email found — Generate badge (amber, clickable). Clicking derives firstname@domain.com from stored website, populates field, badge becomes ✦ Generated (cyan). No layout shift.
  • Footer: Skip prospect (left) · Save · Approve → (right)
  • Approve/Skip from drawer has same effect as row buttons
Prospects: hs · mj · sc hs: howard@youstudio.com (no email) mj: mark@latitudeagency.co.uk (no email) sc: sophie@foldstudio.co.uk (has email)

Tab 2 — Drafts

  • Header row: draft count + "Send all" button (opens send drawer in bulk mode)
  • 4 draft rows: James Mitchell, Sarah Thompson, Marcus Peters, Helen Clarke
  • Each row: avatar · name · subject line preview · Edit button · Send button
  • Send button opens the send drawer for that prospect

Send drawer (right panel)

  • Slides in from right (520px wide)
  • Two modes: single prospect send · bulk send all
  • Single mode: shows prospect name + email, subject line (editable), body (editable), attachment zone
  • 2-step confirm: "Send email" button → confirmation message appears → "Confirm send" button
  • On confirm: toast shows email address, drawer closes, row fades from list
  • Bulk mode: shows count of drafts, same 2-step confirm

Tab 3 — LinkedIn

  • Filter pill bar: No connection · Connection sent · Connection accepted · Unable to connect · Chose not to connect
  • Prospect rows with LinkedIn status dropdown per row
  • LinkedIn button on each row: opens LinkedIn people search for that person in new tab AND copies the LinkedIn message template (with {{first_name}} merged) to clipboard
  • Clipboard copy shows cyan confirmation bar: "✓ LinkedIn message copied to clipboard"

Tab 4 — Sent

  • Master record of all emailed prospects
  • Each row has a 14-option status dropdown

Tab 5 — Follow-up

  • Subtitle: "14 prospects due for follow-up. Sequences pause automatically when they reply."
  • Two groups: Due today · Upcoming
  • Each row: avatar · name · subtitle · days since sent · Follow-up button
  • Follow-up button opens compose toast/action

Tab 6 — Nurture

  • Subtitle: "Prospects parked for re-contact. We'll surface them when the time comes."
  • Prospect rows with re-contact date and notes

Tab 7 — Conversations

  • Subtitle: "Active replies and meetings. Move each one forward or close it out."
  • Prospect rows with reply snippet and action buttons

Tab 8 — Bounced

  • Amber banner: "3 emails bounced in the last 24 hours — We've found alternative addresses to try."
  • Group: "Bounced — alternatives available"
  • 3 rows: Howard Sullivan (try alowe@creativelab.com), Dan Hughes (try d.hughes@agencyco.io), Laura Webb (no alternative found)
  • Each row: Retry button (amber) · Skip button

LinkedIn Message Config Modal

  • Opened via "LI message" button in campaign header
  • Centre-screen modal overlay (not a side drawer)
  • Textarea with default message pre-filled: "Hi {{first_name}}, I wanted to connect to introduce myself and the fractional new business support I help agencies with..."
  • Supports merge tags: {{first_name}} · {{company}}
  • Save button stores message for session · Cancel closes without saving
  • Message is per-campaign (stored in JS for this wireframe)
