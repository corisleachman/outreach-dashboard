Sent Tab V2 — design notes
No dropdowns. State is read-only and derived from data. The user never sets a status manually.
Left accent bar on state-replied, state-action-needed, state-failed. Invisible on passive rows — only draws the eye when something needs attention.
Pulsing dot on in-sequence rows signals the system is running. Static dot on terminal states (replied, failed, complete).
Section dividers separate "Needs your attention" from "In progress". Attention rows are always shown first.
CTA stopPropagation. Clicking a CTA button does not open the drawer. Clicking anywhere else on the row does.
"Decide" CTA on sequence-complete opens a small contextual menu: Nurture / Mark as closed / Create task. Not a dropdown — a small popover.
Microcopy guide: State labels are always present tense ("Follow-up scheduled", "Replied", "Email failed") with a time reference in the context line below.
