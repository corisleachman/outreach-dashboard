Nurture Tab V2 — design notes
Two sections. "Ready to re-engage" (timing has passed, action recommended) sits above "Parked" (still waiting). The split makes it immediately clear where attention is needed without requiring the user to read every row.
Reason chip (.nreason) is human-readable text, not a system code. It explains why the prospect is here. Not editable in v1.
Ready rows get amber tint + amber CTA. Parked rows are neutral — no colour urgency, just timing context.
Task chip only appears when an open task exists for this prospect. Click → opens task in drawer. Prevents duplicate task creation.
Hint text (.nhint) is optional, subtle, italic. Gives a single-line human steer without being prescriptive.
Re-engage CTA moves the prospect back to Review (or directly to Drafts if a message template exists). Does not trigger row click — stopPropagation.
Future exploration (v2+): contextual signals ("role change detected", "company news") could appear as a subtle extra line below the reason chip. Not in v1.
