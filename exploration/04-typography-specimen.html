Font Family Comparison

Space Grotesk

Display font. Bold, condensed, editorial. 70s Swiss energy meets contemporary design galleries. Used for hero headlines, statement sentences, and metric values.

Bold type Takes space Commands attention

DM Sans

Supporting font. Clean, geometric, lets the display font shine. Used for labels, body copy, secondary information, and all supporting text. Never competes.

Clean supporting Reads fast Gets out of the way
Usage Rules

Space Grotesk (Display)

  • ✓ Hero headlines (96px+)
  • ✓ Statement sentences (32px–56px)
  • ✓ Metric values (48px–64px)
  • ✓ Section headings (24px–32px)
  • ✓ Always bold or semibold
  • ✗ Never body copy
  • ✗ Never at tiny sizes (<24px)

DM Sans (Supporting)

  • ✓ Body copy (16px)
  • ✓ Supporting text (14px)
  • ✓ Labels & metadata (12px)
  • ✓ Captions & tags (11px)
  • ✓ 400–700 weight range
  • ✗ Never for hero headlines
  • ✗ Never bold (max 700)
